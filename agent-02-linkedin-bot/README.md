## What it does

A job-search assistant built specifically for CS students hunting tech internships on LinkedIn. Paste in your background and a job description, and Gemini Pro generates a targeted LinkedIn search strategy, a personalized cold outreach message, and a breakdown of the JD — including a flag if the role offers visa sponsorship for international students.

## Features

- Search strategy generator — tailored LinkedIn queries and recruiter approach based on your profile
- Cold outreach writer — personalized connection request and follow-up message drafts
- JD analyzer — key requirements summary with international student sponsorship detection

## How to use

1. Visit the [live demo](https://hhanng.github.io/agent-02-linkedin-bot/)
2. Fill in your background (major, year, skills, target role)
3. Paste a job description and select the tool you need — strategy, outreach, or analysis

   > Add your Gemini API key at [aistudio.google.com](https://aistudio.google.com)
   > for the AI-powered features
