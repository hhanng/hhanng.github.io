## What it does

An interactive coaching tool for anyone who wants to get better at talking to AI. Type a prompt, and Gemini Pro scores it, explains exactly what's weak, rewrites it professionally, and shows a live comparison of the before/after outputs. A built-in techniques library teaches the key prompting patterns — role prompting, chain-of-thought, few-shot, and more — with runnable examples.

## Features

- Prompt evaluator — scores your prompt and explains specific weaknesses
- Prompt improver — rewrites your prompt professionally with a before/after comparison
- Techniques library — interactive examples of core prompting patterns you can run immediately

## How to use

1. Visit the [live demo](https://hhanng.github.io/agent-03-prompt-coach/)
2. Type any prompt into the evaluator and click **Analyze** to see your score and feedback
3. Click **Improve** to get a professional rewrite, then explore the Techniques library to learn the patterns

   > Add your Gemini API key at [aistudio.google.com](https://aistudio.google.com)
   > for the AI-powered features
