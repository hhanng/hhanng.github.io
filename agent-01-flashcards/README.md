## What it does

A study tool built for 9th graders reading Orson Scott Card's *Ender's Game*. It ships with 40 hand-crafted flashcards covering Characters, Themes, Plot, and Vocabulary, and includes an AI Generate mode powered by Gemini Pro that creates custom cards on any topic from the novel. Cards flip with a smooth 3D animation and can be shuffled or filtered by category.

## Features

- 40 pre-loaded cards across 4 categories: Characters, Themes, Plot, Vocabulary
- 3D flip animation — click any card to reveal the answer
- Category filters and shuffle mode for flexible studying
- AI Generate mode — describe a topic and Gemini Pro writes a brand-new card

## How to use

1. Visit the [live demo](https://hhanng.github.io/agent-01-flashcards/)
2. Pick a category (or leave it on All) and click any card to flip it
3. Hit **Shuffle** to randomize the deck
4. Switch to **AI Generate**, type a topic from the book, and click Generate

   > Add your Gemini API key at [aistudio.google.com](https://aistudio.google.com)
   > for the AI-powered features
